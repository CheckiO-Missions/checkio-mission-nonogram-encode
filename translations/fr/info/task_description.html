Cette mission est à l'opposé: vous disposez déjà d'une image binaire, formée de chaînes de caractères de longueur égale, où une cellule vide en arrière-plan contient un espace blanc et une cellule d'image contient 'X'.

Votre objectif est de créer un nonogramme basé sur l'image: écrire des indices numériques pour résoudre l'image comme si elle était cachée. Votre fonction doit renvoyer {% if is_js %}un tableau de deux tableaux{% else %}une liste de deux listes{% endif %}. Tous les tableaux {% if is_js %}{% else %} listes{% endif %} des indices de colonnes, ainsi que des indices de des lignes, doivent avoir la même "profondeur" (complétée par 0). Voyons un exemple:
